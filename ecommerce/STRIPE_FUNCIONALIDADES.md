# Stripe: Selección de Funcionalidades

## Pantalla: "¿Cómo quieres empezar?"

Cuando Stripe te pregunta qué funcionalidades necesitas, aquí está la guía para este proyecto:

## ✅ Opción Correcta para Este Proyecto

### Marca SOLO esta opción:

✅ **"Pagos no recurrentes"** (Non-recurring payments)

**Por qué:**
- Este proyecto permite comprar tokens con tarjeta de crédito
- Cada compra es un pago único
- No necesitas suscripciones ni pagos recurrentes
- Es la opción más simple y adecuada

## ❌ No Necesitas Estas Opciones

Puedes dejar sin marcar:

- ❌ **Pagos recurrentes** (Recurring payments)
  - Solo si tuvieras suscripciones mensuales/anuales
  
- ❌ **Facturas** (Invoices)
  - Stripe puede generar facturas, pero no es necesario para este proyecto
  
- ❌ **Plataforma o marketplace**
  - Solo si fuera una plataforma multi-vendor
  
- ❌ **Cobro de impuestos** (Tax collection)
  - Stripe puede calcular impuestos automáticamente, pero no es necesario ahora
  
- ❌ **Pagos en persona** (In-person payments)
  - Solo para terminales físicas
  
- ❌ **Emisión de tarjetas** (Card issuance)
  - Solo si emitieras tarjetas físicas

## Pasos a Seguir

1. ✅ Verifica que **"Pagos no recurrentes"** esté marcada
2. ✅ Asegúrate de que las demás opciones estén desmarcadas
3. ✅ Haz clic en **"Continuar"**

## ¿Y Si Ya Marcaste Otras?

No hay problema. Puedes:
- Continuar igualmente (no afecta negativamente)
- O volver atrás (botón "← Atrás") y desmarcar las que no necesitas

Stripe solo personaliza las recomendaciones según tus selecciones, pero siempre puedes usar todas las funcionalidades.

## Resumen Rápido

```
✅ Pagos no recurrentes  ← MARCA ESTA
❌ Todo lo demás         ← DEJA SIN MARCAR
```

Luego haz clic en **"Continuar"** y listo.

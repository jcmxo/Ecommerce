# Iconos de la Extensión

La extensión requiere iconos en los siguientes tamaños:
- `icon16.png` - 16x16 píxeles
- `icon48.png` - 48x48 píxeles  
- `icon128.png` - 128x128 píxeles

Puedes crear estos iconos usando cualquier herramienta de diseño. Los iconos deben colocarse en la carpeta `dist/` después del build.

Para desarrollo rápido, puedes usar iconos placeholder o generar iconos simples con herramientas online como:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Los iconos deben tener el mismo diseño pero en diferentes tamaños para que se vean bien en todos los contextos de Chrome.
